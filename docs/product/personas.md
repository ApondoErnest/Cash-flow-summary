## Owner (Super Administrator)

**Context:** Owns multiple inspection centers in one organization.

**Goals:** Oversee each center in turn; approve revisions; manage users and centers; receive WhatsApp alerts.

**Frustrations:** Accidental cross-center data mixing; unclear which center is active; duplicate revenue in spreadsheets.

**Key flows:** Login → Center Selection → active center dashboard; verify/import; switch center; admin centers/users.

**Constraints:** One active working center at a time; no All Centers operational view.

---

## Center Manager

**Context:** Runs one assigned center day to day.

**Goals:** Import daily CSV; monitor center dashboard; submit corrections when source data changes; review anomalies.

**Frustrations:** Complex multi-screen tools; fear of importing wrong center's file.

**Key flows:** Fixed-center import card; import history; correction submission; center reports and record search.

**Constraints:** Cannot switch center; cannot approve own revisions.

---

## Cashier

**Context:** Front desk; uploads cashier statement at end of day.

**Goals:** Quick verify → import with clear footer totals; see if upload succeeded; download errors if any.

**Frustrations:** Unclear whether duplicates were ignored; too many screens.

**Key flows:** Compact dashboard; verify/import/reject; recent imports; error report download.

**Constraints:** Minimal navigation; no reports beyond recent activity; no user admin.
